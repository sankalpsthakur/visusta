# Visusta Regulatory Reports — Build Instructions

## Source Files

| File | Purpose |
|------|---------|
| `generate_charts.py` | Generates all 6 matplotlib charts (run first) |
| `build_monthly_report.py` | Builds the 8-page Monthly Impact Report PDF |
| `build_quarterly_brief.py` | Builds the 11-page Quarterly Strategic Brief PDF |
| `charts/` | Pre-generated chart PNGs (output of generate_charts.py) |

## Prerequisites

```bash
pip install reportlab matplotlib
```

## Build Steps

```bash
# 1. Generate charts
python3 generate_charts.py

# 2. Build Monthly Report
python3 build_monthly_report.py

# 3. Build Quarterly Brief
python3 build_quarterly_brief.py
```

## Output

- `Visusta_Monthly_Impact_Report_Feb2026.pdf` (8 pages, ~268 KB)
- `Visusta_Quarterly_Strategic_Brief_Q1_2026.pdf` (11 pages, ~311 KB)

## Customization

- **Brand colors**: Edit the `C_*` hex constants at the top of each build script
- **Content**: Modify the `build_content()` function in each script
- **Charts**: Edit `generate_charts.py` — all chart functions are standalone
- **Page layout**: Adjust `Frame` definitions in `build_pdf()` for margins/spacing

## Architecture

Both reports use ReportLab's `BaseDocTemplate` with dual `PageTemplate`s:
- `cover` template: Full-bleed dark green cover page
- `content` template: Branded header/footer with page numbering

Reusable components: `pro_table()`, `callout_box()`, `status_badge()`
